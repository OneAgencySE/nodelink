# nodelink
A minimal blockchain built in Node JS.

## Purpose
To get a basic understanding of the distributed blockchain concept by building one.

## Installation

`git clone https://github.com/OneAgencySE/nodelink`

`npm install`

## Run

`npm start`

`http://localhost:3001/interface`
